# ZkNoid L1 contracts

ZkNoid uses interplay of mina mainnet and protokit networks. This repository contains L1 contracts of the ZkNoid infrastructure

## How to build

```sh
npm run build
```

## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```
